# LibraryProject

    도서관 프로젝트
   ================
   
   개인 프로젝트 입니다.
   -------------------
  1. 평소에 책에 관심이 많아서 직접 도서에 관한 사이트를 만들어봤습니다.
  2. 학원 과정중에 배운 내용을 토대로 라이브러리를 적극 활용하여 제작하였습니다.
  3. 일반적인 사이트에서 구현할 수 있는 점은 대부분 구현해놨습니다.
